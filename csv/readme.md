# CSV_TO_JSON.SH

## Uso
```shell
chmod +x ./csv_to_json.sh
```

```shell
./csv_to_json.sh ~/input.csv ~/output.json false
```

1) Input de arquivo CSV -> `~/input.csv`

2) Output de arquivo JSON -> `~/output.json` 

3) Double quote ou single quote (default == double quote) -> `false`
