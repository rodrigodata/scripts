# CSV_TO_JSON.SH

## Uso
```shell
chmod +x ./csv_to_json.sh
```

```shell
./csv_to_json.sh ~/input.csv ~/output.json false
```

input de arquivo CSV -> `~/input.csv`

output de arquivo JSON -> `~/output.json`

double quote ou single quote(default == double quote) -> `false`
